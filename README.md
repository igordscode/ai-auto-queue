# 🚀 Queue Master LITE
### Bulk Prompting for AI - Plug & Play

**Queue Master LITE** is a lightweight Chrome extension that automates your AI workflows. Load hundreds of prompts, sit back, and watch it work. 

---

## 🔥 Why Queue Master LITE?

- **Zero Setup:** No Python, no servers, no API keys needed.
- **Auto-Advance:** Automatically detects when the AI finishes and sends the next prompt.
- **Auto-Compile:** Instead of 50 separate files, download **ONE single Markdown file** with your entire session history organized.
- **Custom Filenames:** Name your output file (e.g., `MyBook_Chapter1`) directly in the panel.
- **Smart Logic:** Automatically clicks "Continue Generating" for long responses.

---

## 🛠️ Installation

1. Download or Clone this repository.
2. Open Chrome/Edge and go to `chrome://extensions`.
3. Enable **Developer Mode** (top right).
4. Click **Load Unpacked** and select the `ai-auto-queue` folder.

---

## 🚀 How to Use

1. **Open AI Chat:** Go to ChatGPT or Gemini.
2. **Load Prompts:** Paste your list of prompts (separated by empty lines).
3. **(Optional) Name Your File:** Type a name like `Project_Alpha` in the input box.
4. **Check "Auto-avançar":** This automates the flow.
5. **Click "▶ Iniciar":** The extension works through the queue.
6. **Download:** At the end, a single `.md` file with all prompts and responses will be downloaded automatically.

---

## 📂 Project Structure
- `content.js`: Core logic (Queue management + Browser-based saving).
- `styles.css`: UI Styling.
- `manifest.json`: Extension configuration.
