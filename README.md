# 🚀 Queue Master PRO
### The Ultimate AI Automation & Bulk Prompting Suite

**Queue Master PRO** turns your favorite AI chat interface (ChatGPT, Gemini, etc.) into a powerful production line. Load hundreds of prompts, sit back, and watch it work.

## 🔥 Why Queue Master PRO?

- **Zero Setup:** No Python, no servers, no API keys needed. Just install and run.
- **Auto-Advance:** Automatically detects when the AI finishes and sends the next prompt.
- **Auto-Compile:** Instead of 50 separate files, download **ONE single Markdown file** with your entire session history organized.
- **Smart Logic:** Automatically clicks "Continue Generating" for long responses.
- **Multi-Language:** Auto-detects your language (English, Portuguese, Spanish, French, German, Chinese).
- **Modern UI:** Draggable, minimizable, and designed to stay out of your way.

---

## 🛠️ Installation

1. Download or Clone this repository.
2. Open Chrome/Edge and go to `chrome://extensions`.
3. Enable **Developer Mode** (top right).
4. Click **Load Unpacked** and select the `ai-auto-queue` folder.

---

## 🚀 How to Use

1. **Open AI Chat:** Go to ChatGPT or Gemini.
2. **Load Prompts:** Paste your list of prompts (separated by empty lines).
3. **(Optional) Name Your File:** Type a name like `My_Book_Chapter1`.
4. **Check "Auto-advance":** This automates the flow.
5. **Click "▶ Start":** The extension works through the queue.
6. **Download:** At the end, a single `.md` file with all prompts and responses will be downloaded automatically.
